# 皮皮虾（纯净版）

一个用于皮皮虾 App 的越狱插件，提供去广告、视频倍速播放、无水印下载等功能。

## 功能特性

- ✅ **去除启动页广告** - 跳过应用启动时的广告页面
- ✅ **去除视频列表广告** - 清理视频流中的广告内容
- ✅ **去除评论区广告** - 移除视频详情页评论区的广告
- ✅ **无水印视频下载** - 支持下载无水印的原版视频
- ✅ **无水印图片下载** - 支持下载评论区无水印图片
- ✅ **视频倍速播放** - 支持 0.75x、1.0x、1.25x、1.5x、2.0x 倍速播放
- ✅ **插件设置界面** - 提供便捷的设置入口，可自定义播放倍速

## 系统要求

- iOS 7.0 及以上版本
- 已越狱设备
- 已安装 [Theos](https://theos.dev/) 开发环境（仅编译时需要）

## 下载安装

### 方式一：通过 Cydia/Sileo 安装（推荐）

1. 添加源（如果有提供源地址）
2. 搜索 "ppxtweak" 或 "皮皮虾"
3. 点击安装

### 方式二：手动安装

1. 下载 `.deb` 安装包
2. 使用 Filza 或通过 SSH 安装：
   ```bash
   dpkg -i com.zy.ppxtweak_*.deb
   ```
3. 重启 SpringBoard 或使用 `killall SpringBoard` 命令

### 方式三：从源码编译

```bash
# 克隆仓库
git clone https://github.com/zy364601065/ppxtweak.git
cd ppxtweak/ppxtweak

# 编译
make

# 打包
make package

# 安装
make install
```

## 使用说明

1. 安装插件后，打开皮皮虾 App
2. 首次使用会弹出更新提示，可选择"立即前往"进入插件设置
3. 在皮皮虾设置页面顶部会显示"插件设置"入口
4. 在插件设置中可以：
   - 开启/关闭视频倍速播放功能
   - 选择默认播放倍速（0.75x、1.0x、1.25x、1.5x、2.0x）
   - 查看作者信息
